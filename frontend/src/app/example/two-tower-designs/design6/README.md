# Two-Tower Design 6

이 구현체는 **Gemini 3.5 Flash** 에이전트에 의해 생성되었습니다.

## 특징
1. **다크 테마 & 글래스모피즘(Glassmorphism)**: 고급스럽고 현대적인 UI 디자인
2. **모의 추천 연산 엔진**: 10가지 설문 문항 및 업종별 가중치를 활용해 실시간으로 9가지 성향 점수를 산출하고, 모의 서울 상권 데이터셋 중 가장 적합도가 높은 Top 5 상권을 동적으로 추천
3. **공유 가능한 동적 라우팅**: `design6/[res]` 동적 파라미터를 사용해 결과 페이지 URL을 공유할 수 있으며, 공유 코드를 디코딩하거나 `localStorage`를 조회해 결과를 원활히 복구
4. **저장 및 토스트 알림**: 가상으로 로그인 상태와 설문 결과 저장을 지원하는 인터랙티브 요소 포함
