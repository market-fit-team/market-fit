# Two-Tower Design 7 (Zinc Minimalism)

이 구현체는 **Gemini 3.5 Flash** 에이전트에 의해 생성되었습니다.

## 특징
1. **Zinc 계열 미니멀리즘 디자인**: 자극적인 원색이나 화려한 보라색 그라데이션을 모두 제거하고, 얇은 보더 라인과 차분한 Zinc(회색/Neutral) 톤의 정제된 미니멀리즘 테마를 구축했습니다.
2. **라이트/다크 모드 전면 지원**: 컴포넌트 레벨에서 브라우저 및 OS 테마 환경(`(prefers-color-scheme: dark)`)을 자동 감지하며, 헤더 우측의 달/해 모양 토글 버튼을 활용해 동적으로 즉각적인 테마 전환이 가능합니다.
3. **독립 모의 매칭 엔진**: 10가지 성향 질문과 예비창업가 선호 업종 가중치를 산출하여, 가장 유사도가 높은 상주인구/직장인/역세권 등 서울의 5대 상권을 도출합니다.
4. **공유 코드 복구**: 공유용 고유 코드를 통하여 접속 시 로컬스토리지가 초기화된 경우에도 해시 매핑을 기반으로 완벽히 동일한 결과 화면을 복원합니다.
